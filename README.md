# Caixa Eletrônico - Java 🏧

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos (POO). 
Trata-se de um sistema de simulação de Caixa Eletrônico com interface gráfica (GUI).

## 🚀 Funcionalidades
* **Efetuar Saque:** Algoritmo que calcula e entrega o menor número de cédulas possíveis, priorizando as de maior valor.
* **Relatório de Cédulas:** Visualização do estoque atual do caixa por tipo de nota.
* **Valor Total:** Cálculo em tempo real de todo o dinheiro disponível.
* **Reposição de Cédulas:** Módulo do administrador para abastecer o caixa.
* **Cota Mínima:** Trava de segurança que impede saques caso o saldo atinja um limite configurado.
* **Extrato de Operações:** Histórico das transações realizadas na sessão.

## 🛠️ Tecnologias Utilizadas
* Java
* Java Swing (Interface Gráfica)
* Paradigma de Orientação a Objetos

## ⚙️ Como executar
Para testar o projeto sem precisar compilar o código, basta baixar o arquivo `.jar` na aba **Releases** ao lado e executá-lo com o Java instalado na máquina.
